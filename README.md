# 🚗 Car Rental Full-Stack Application

A modern, high-performance car rental platform built with **Next.js**. This application handles everything from the user interface to the backend API logic and database interactions within a single unified framework.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Frontend:** React, Tailwind CSS, Lucide React (Icons)
- **Backend:** Next.js API Routes (Serverless Functions)
- **Database:** MySQL (via XAMPP/Local Instance)
- **Styling:** PostCSS & Tailwind

---

## 🚀 Key Features

- **Full-Stack Integration:** Backend logic handled via `src/app/api` routes.
- **Categorized Fleet:** Dedicated sections for **VIP**, **Mid-range**, and **Economic** vehicles.
- **Responsive Design:** Optimized for mobile, tablet, and desktop experiences.
- **Dynamic Routing:** Specific car details and categories powered by Next.js dynamic segments.

---

## 📦 Getting Started

To run this project locally, follow these steps:

### 1. Prerequisites
- **Node.js** (Latest LTS version)
- **XAMPP** (For the MySQL database)

### 2. Database Setup
1. Start **MySQL** in your XAMPP Control Panel.
2. Go to `http://localhost/phpmyadmin`.
3. Create a database named `car_rental`.
4. Import the provided `.sql` file found in the `/db` folder of this project.
